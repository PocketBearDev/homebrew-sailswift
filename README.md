# Homebrew Tap for Sailswift

This is a [Homebrew](https://brew.sh/) tap for [Sailswift](https://github.com/proverbiallemon/Sailswift), a native macOS mod manager for Ship of Harkinian.

## Installation

```bash
brew tap PocketBearDev/sailswift
brew install --cask sailswift
```

## Updating

```bash
brew upgrade --cask sailswift
```

## Uninstalling

```bash
brew uninstall --cask sailswift
brew untap PocketBear/sailswift
```

## About Sailswift

Sailswift is a native macOS mod manager for Ship of Harkinian, built with Swift and SwiftUI. Features include:

- Mod management with enable/disable/delete
- Load order control with drag-and-drop
- GameBanana integration for browsing and downloading mods
- One-click install via `shipofharkinian://` URL scheme
- Automatic updates via Sparkle

For more information, visit the [Sailswift repository](https://github.com/proverbiallemon/Sailswift).
